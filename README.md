# 📦 Delivery Problem Resolution

Painel interativo de resolução de problemas de entrega para operações Amazon Last Mile (EDSP).

## 🎯 Objetivo

Guia visual com passo a passo para motoristas e equipe de campo resolverem os 11 problemas mais comuns durante entregas.

## 📋 Categorias

| # | Problema | Arquivo |
|---|----------|---------|
| 1 | Problema com GPS | `gps.html` |
| 2 | Pacote Danificado | `pacote-danificado.html` |
| 3 | Pacote Não Listado no Dispositivo | `pacote-nao-listado.html` |
| 4 | Pacote Físico Ausente | `pacote-fisico-ausente.html` |
| 5 | Localizar Código OTP | `localizar-otp.html` |
| 6 | Não Foi Possível Acessar o Local | `nao-possivel-acessar.html` |
| 7 | Cliente Reprogramou a Entrega | `reprogramou-entrega.html` |
| 8 | Endereço Não Localizado | `endereco-nao-localizado.html` |
| 9 | Cliente Se Mudou de Endereço | `cliente-mudou.html` |
| 10 | Problemas Climáticos | `problemas-climaticos.html` |
| 11 | Empresa Fechada | `empresa-fechada.html` |
| 12 | Cliente Não Encontrado | `cliente-nao-encontrado.html` |

## 🚀 Como usar

1. Abra `index.html` no navegador
2. Clique na categoria do problema
3. Siga as etapas ilustradas

## 📱 Responsivo

Funciona em desktop, tablet e celular.

## 🖼️ Imagens

As imagens `step1.png` a `step37.png` são prints das telas do app de entrega. Devem estar na mesma pasta que os arquivos HTML.

## 🛠️ Tecnologia

- HTML/CSS/JS puro (zero dependências)
- Sem backend necessário
- Funciona offline após download
